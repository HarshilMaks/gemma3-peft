Nested Settings with Detail Modals

Initial Main Settings List (e.g., 'Account', 'Display & Sound', 'Notifications & Alerts', 'Privacy & Data')
Chevron/Arrow Indicator (Next to each list item, indicating it opens a sub-page or modal)
Clicking a Main Setting Opens a Full-Page Modal/Drawer
Modal/Drawer Header (e.g., 'Account Settings')
Modal/Drawer 'Back' Button (To return to the main settings list)
Sub-Setting List within Modal (e.g., 'Edit Profile', 'Change Password', 'Two-Factor Authentication')
Clicking a Sub-Setting might open another nested modal or expand inline.
Inline Field Editors or Small Popovers for simple changes within modals.
Confirmation Prompts (Before saving changes or performing critical actions within modals).
